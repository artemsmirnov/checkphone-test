## Config
Create `local.json` in `config` folder. See `default.json` from schema

## Setup

``` bash
# install dependencies
npm install

# bootstrap db schema
npm run db:schema

# bootstrap db seeds
npm run db:seeds

# Run service
npm run service:{phone,auth}

# Run service with autoreload
npm run service:{phone,auth}:dev
```
